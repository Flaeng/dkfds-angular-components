A short description on how to setup and use this library.

Setup

Prerequisites: You have node, npm and angular-cli installed on your computer and know how to start a new angular project.

  1. Install fds in your project by running the following command in the root of the project: npm install dkfds
  2. Setup your styles.scss-file to look something like this: @import '~dkfds/dist/css/dkfds.css';

    $theme-colors: (
      "primary": (
        100: #44831E,
        200: #3C5C22,
        300: #233614
      ),
      "background": (
        100: #f1f1f1
      )
    );

    $theme-color-primary: 'primary-100';
    $theme-color-primary-dark: 'primary-200';
    $theme-color-primary-darker: 'primary-300';
    $header-portal-background-color: 'background-100';
    $header-portal-logo-height: 4.8rem;
    $header-portal-compact-logo-width: 10rem;

    $font-path: '~dkfds/src/fonts/IBMPlexSans/';
    $image-path: '~dkfds/src/img';
    $site-image-path: '~dkfds/src/img';
    $icons-folder-path: '~dkfds/src/img/svg-icons';

    @import '~dkfds/src/stylesheets/dkfds';
  3. Install this libary by running the following command npm install @flaeng/fds-angular
  4. Start using the ngfds, directives and services as you like. All components are prefixed with 
    <fds-(...)
    i.e. <fds-card></fds-card>
    and all directives are prefixed with 
    <div fds-(...)
    i.e. <div fds-tooltip="This is a tooltip"></div>
    and all services are called 
    Fds(...)
    i.e. FdsToaster

Versioning

When installing

How to decide what version of this library to install?
First: Check what major-version of Angular you are using in your project.
Second: Check what major-version of FDS you are using in your project.
Now select the version from this formula: [AngularMajorVersion].[FdsMajorVersion].X.X where X and X are as large as possible.
i.e. if you use Angular 11 and FDS version 8 select 11.8.1 over 13.8.0 and 11.8.0

When installing this library use the ~(tilde)-character to tell npm to always use this major and minor version (the versions that match your Angular and FDS versions).
I.e. 

npm install @flaeng/fds-angular@~13.8.1

or for Angular 11

npm install @flaeng/fds-angular@~11.8.1

When upgrading Angular

When upgrading Angular remmeber to update this library as well, as the same package does not support multiple major-versions of Angular.
